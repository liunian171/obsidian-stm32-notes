# 硬件模块信息

## 电源

### 2S 锂电池
- **电压**: 7.4V (标称)
- **尺寸**: 75.7 × 33.2 × 18.3 mm (平放)
- **用途**: 总电源，分流为逻辑电和驱动电

### 5V 稳压模块
- **输入**: 7.4V (来自2S电池)
- **输出**: 5V
- **用途**: 逻辑供电 → ESP32-S3

### 7.4V 驱动轨
- **输入**: 7.4V (来自2S电池直出)
- **用途**: 驱动供电 → PCA9685 ×2

## 主控

### ESP32-S3
- **供电**: 5V (逻辑)
- **接口**: I2C → 两路PCA9685
- **框架**: Arduino (PlatformIO)
- **频率**: 240MHz

## 舵机驱动

### PCA9685 #1
- **I2C 地址**: 0x40
- **驱动**: 步态A组 (腿1,3,5) — 9路舵机
- **PWM**: 50Hz, 0-180° → 102-512

### PCA9685 #2
- **I2C 地址**: 0x41
- **驱动**: 步态B组 (腿2,4,6) — 9路舵机
- **PWM**: 50Hz, 0-180° → 102-512

## 舵机

### 舵机 ×18
- **每腿**: 3个 (髋A / 膝B / 踝C)
- **关节范围**: A=80(后)~100(前), B=90(直)~100(弯), C=180
