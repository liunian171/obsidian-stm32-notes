# DaliangAuto 项目总览

## 项目定位
大亮Auto (DaliangAuto) 是一套**基于单目视觉的自动跟随小车系统**，涵盖从数据采集、模型训练到嵌入式部署的完整 AI 工程链路。项目定位为 AI 动手学习项目。

## 项目组成（6个子项目）

| 子项目 | 角色 | 平台/技术 | 核心职责 |
|--------|------|-----------|----------|
| **vision-follow-car** | 文档导航枢纽 | Markdown | 项目总说明、架构文档、各子仓库链接 |
| **vision-follow-car-mcu** | 底层控制固件 | STM32F103RCT6 (Bare-metal C) | 转向/油门/刹车控制、模式切换、UART通信 |
| **vision-follow-car-jetson** | 车端推理应用 | Jetson Orin Nano (Python) | CSI摄像头采集、神经网络推理、控制指令输出 |
| **vision-follow-car-training** | 服务器端训练 | PyTorch / GPU Server | 数据处理、模型训练、验证、导出 |
| **Dliang-Open-ERP** | 库存/进销存管理 | FastAPI + SQLite | BOM管理、库存跟踪、采购销售订单 |
| **Aluminum-profile-go-kart** | 机械结构设计 | 铝型材 + STEP图纸 | 物理底盘、悬架、转向机构 |

## 数据流

```
训练阶段:
  MCU(50Hz控制数据) ──UART──→ Jetson ──同步保存──→ 图像+控制配对数据 ──→ 服务器训练

推理阶段:
  CSI摄像头 ──→ Jetson(模型推理) ──UART──→ MCU ──→ 转向/油门/刹车执行
```

## 系统架构总览

```
┌─────────────────────────────────────────────────────────────────────┐
│                        离线训练 (Training)                           │
│  vision-follow-car-training                                         │
│  原始数据 → 清洗 → 训练(ResNet18+GRU) → 导出模型                    │
└────────────────────────────┬────────────────────────────────────────┘
                             │ 模型权重
                             ▼
┌─────────────────────────────────────────────────────────────────────┐
│                        在线推理 (Jetson Orin Nano)                   │
│  vision-follow-car-jetson                                           │
│  CSI摄像头 → 预处理 → 5帧缓冲 → 推理 → 控制指令                    │
└────────────────────────────┬────────────────────────────────────────┘
                             │ UART (22字节帧, CRC16)
                             ▼
┌─────────────────────────────────────────────────────────────────────┐
│                        底层控制 (STM32F103RCT6)                      │
│  vision-follow-car-mcu                                              │
│  4种模式: 手动/遥控/训练/跟随 → 转向伺服/油门PWM/刹车电缸          │
└─────────────────────────────────────────────────────────────────────┘
```

## 关键技术栈

| 领域 | 技术 |
|------|------|
| MCU | STM32F103RCT6, HAL库, Bare-metal, PWM/ADC/IC/CAN/UART/I2C |
| 视觉AI | ResNet18, GRU, PyTorch, OpenCV, GStreamer |
| 通信 | CAN 2.0, UART(115200bps), 自定义22字节帧协议, CRC16-MODBUS |
| 后端 | FastAPI, SQLAlchemy, SQLite, Jinja2 |
| 机械 | 铝型材框架, 铝型材框架, 无刷轮毂电机, 直流伺服, 线刹 |

## 设计亮点
1. **端到端 AI 链路完整闭环**: 从数据采集到实车执行一步不落
2. **三组件架构分离**: 训练/推理/控制各自独立，接口清晰
3. **UART自定义协议**: 22字节帧 + CRC16校验，适合嵌入式实时通信
4. **4种工作模式**: Manual→Remote→Training→Follow，逐步递进
5. **模仿学习范式**: 人工驾驶采集数据 → 端到端行为克隆
