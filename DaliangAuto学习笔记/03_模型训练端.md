# 模型训练端深度分析 — Imitation Learning

## 项目定位
服务器端训练管线，使用模仿学习(Behavioral Cloning)让模型学习人类驾驶行为。

## 模型架构: ResNet18 + GRU

```
输入: images [B, 5, 3, 224, 224]      speeds [B, 5, 1]
             │                               │
             ▼                               ▼
      ResNet18(去除fc)                SpeedEncoder(MLP)
      [B*5, 512, 1, 1]               [B*5, 1]→[B*5, 16]
             │                               │
             ▼                               ▼
      feat [B, 5, 512]          speed_feat [B, 5, 16]
             │                               │
             └─────────→ concat ←────────────┘
                             │
                             ▼
                   combined [B, 5, 528]
                             │
                             ▼
                   GRU(hidden=256) ← 时序融合
                             │
                             ▼
                    Head MLP(256→128→4)
                             │
                             ▼
              [steer, throttle, brake_logit, target_valid_logit]
```

### 各组件参数

| 组件 | 层结构 | 输入→输出 | 参数量 |
|------|--------|-----------|--------|
| Backbone | ResNet18(去fc) | [B*5,3,224,224]→[B,5,512] | ~11M |
| SpeedEncoder | Linear(1→16→16) + ReLU | [B,5,1]→[B,5,16] | ~288 |
| GRU | 单层, hidden=256, batch_first | [B,5,528]→[B,256] | ~800K |
| Head | Linear(256→128→4) + ReLU | [B,256]→[B,4] | ~33K |

## 训练流程

### Loss设计 (加权多任务)

| 输出 | Loss | 权重 | 说明 |
|------|------|------|------|
| steer | SmoothL1Loss | 2.0 | 回归，允差鲁棒 |
| throttle | SmoothL1Loss | 1.0 | 回归 |
| brake | BCEWithLogitsLoss | 2.0 | 二分类 |
| target_valid | BCEWithLogitsLoss | 3.0 | 二分类(最重要) |

### 采样策略

```
main : no_target = 8 : 1 (加权采样)
WeightedRandomSampler 确保两类平衡
train/val = 70%/30% 时序划分(无时间泄漏)
```

## 学习要点

1. **模仿学习范式** — 端到端行为克隆，从人类示范学习
2. **时序建模** — GRU处理5帧连续图像，捕捉时序信息
3. **多模态融合** — 图像特征 + 速度特征拼接
4. **多任务Loss** — 不同输出不同Loss函数+权重，target_valid权重最高
5. **类别平衡** — 有人/无人按8:1加权采样
6. **时序划分** — 按时间先后拆分训练/验证，防止数据泄漏
7. **CSV+图像配对** — 经典的数据集组织方式
8. **轻量数据增强** — 仅亮度对比度调整，不过度增强

### 为什么用5帧?
- 0.5秒时间窗口 (@10fps录制)
- 足够捕捉转向趋势和速度变化
- 不过度增加模型复杂度
