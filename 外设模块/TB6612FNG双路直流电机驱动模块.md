# TB6612FNG 双路直流电机驱动模块

## 概述

TB6612FNG 是 Toshiba 生产的双通道 H 桥电机驱动芯片，可独立控制两路直流电机。相比传统的 L298N，它体积更小、效率更高（MOSFET 管压降低），且无需额外的散热片。

![[图片-TB6612FNG 双路直流电机驱动模块.png]]
**核心参数：**

| 参数         | 值                 |
| ---------- | ----------------- |
| 工作电压 (VM)  | 2.5V ~ 13.5V      |
| 电机驱动电流     | 单路连续 1.2A，峰值 3.2A |
| 逻辑电压 (VCC) | 2.7V ~ 5.5V       |
| 控制信号电平     | 兼容 3.3V / 5V      |
| 待机电流       | 0.1μA（典型）         |
| 工作模式       | 正转 / 反转 / 刹车 / 停止 |

## 引脚说明

### 电机电源与控制接口

| 引脚   | 类型     | 说明                                                                                                               |
| ---- | ------ | ---------------------------------------------------------------------------------------------------------------- |
| VM   | 电源输入   | 电机供电正极，2.5V~13.5V。接电池或稳压电源。电流能力需满足两路电机同时工作。                                                                      |
| GND  | 电源地    | 电源地。**必须与逻辑电源（VCC）共地。**                                                                                          |
| VCC  | 逻辑电源输入 | 内部控制逻辑供电，2.7V~5.5V。通常接 MCU 板输出的 3.3V 或 5V。                                                                       |
| STBY | 逻辑输入   | **待机控制**（Standby）：高电平 = 正常工作；低电平 = 进入待机（休眠），电机驱动停止，功耗极低。**使用前必须拉高**（可直接接 VCC 或由 MCU GPIO 控制）。      //==可用于急停功能== |

### 电机 A 控制（对应输出 AO1/AO2）

| 引脚 | 类型 | 说明 |
|------|------|------|
| PWMA | 逻辑输入 (PWM) | 电机 A 的 PWM 速度控制。PWM 占空比决定电机转速，频率推荐 10kHz~50kHz。 |
| AIN1 | 逻辑输入 | 电机 A 方向控制引脚 1，与 AIN2 配合决定转向 |
| AIN2 | 逻辑输入 | 电机 A 方向控制引脚 2 |
| AO1 | 输出 (接电机) | 电机 A 的输出端 1 |
| AO2 | 输出 (接电机) | 电机 A 的输出端 2 |

### 电机 B 控制（对应输出 BO1/BO2）

| 引脚 | 类型 | 说明 |
|------|------|------|
| PWMB | 逻辑输入 (PWM) | 电机 B 的 PWM 速度控制 |
| BIN1 | 逻辑输入 | 电机 B 方向控制引脚 1 |
| BIN2 | 逻辑输入 | 电机 B 方向控制引脚 2 |
| BO1 | 输出 (接电机) | 电机 B 的输出端 1 |
| BO2 | 输出 (接电机) | 电机 B 的输出端 2 |

### 其他

| 引脚 | 类型 | 说明 |
|------|------|------|
| GND | 电源地 | 通常模块上会有多个 GND 引脚，全部接至系统共地 |

## 真值表与控制逻辑

| AIN1 | AIN2 | PWMA | AO1/AO2 | 模式 |
|------|------|------|---------|------|
| H | L | PWM | H / L | 正转（顺时针），PWM 调速 |
| L | H | PWM | L / H | 反转（逆时针），PWM 调速 |
| H | H | x | L / L | 刹车（短路制动） |
| L | L | x | 高阻 | 停止（惯性滑行） |
| STBY=L | x | x | 高阻 | 待机（电机断电） |

> 电机 B 同理，将 AIN1/AIN2/PWMA 替换为 BIN1/BIN2/PWMB。

## 典型接线图

```
                    ┌──────────────┐
   MCU 3.3V ────────┤ VCC          │
   电池 7.4V ───────┤ VM           │
                    │              │
   PA0 (PWM) ───────┤ PWMA         ├─── 电机 A (红线)
   PC0 ─────────────┤ AIN1         ├─── 电机 A (黑线)
   PC1 ─────────────┤ AIN2         │
                    │   TB6612FNG  │
   PA1 (PWM) ───────┤ PWMB         ├─── 电机 B (红线)
   PC2 ─────────────┤ BIN1         ├─── 电机 B (黑线)
   PC3 ─────────────┤ BIN2         │
                    │              │
   PB0 ─────────────┤ STBY         │
    └── 拉高或 MCU  │              │
                    │ AO1/AO2 ---- 电机 A
                    │ BO1/BO2 ---- 电机 B
                    └──────────────┘
                          │
                         GND (电池 + MCU 共地)
```

## 注意事项

1. **共地是必须的：** VM 的 GND、VCC 的 GND、MCU 的 GND 必须连接在一起。
2. **STBY 必须拉高**才能工作。可以用 GPIO 控制以实现低功耗待机，或在不需要待机功能时直接接 VCC。
3. **PWM 频率选择：** TB6612FNG 支持较高 PWM 频率（最高 100kHz）。推荐 10kHz~50kHz，低于 1kHz 可能听到线圈啸叫。
4. **电源去耦：** VM 与 GND 之间靠近模块放置 10μF~100μF 电解电容，抑制电机启停时的电压尖峰。
5. **电机电流限制：** 单路持续 1.2A，峰值 3.2A（受热限制，不能长时间处于峰值）。电机堵转或启动瞬间电流可能超过此值，需在软件中做限流或保护。
6. **逻辑电平兼容：** AIN1/AIN2/BIN1/BIN2/PWMA/PWMB/STBY 均为 5V 耐受，3.3V MCU 可直接驱动，无需电平转换。
7. **VS 检测引脚：** 某些模块（非官方）引出 VS 引脚用于检测 VM 电压，查阅具体模块丝印确认。

## 应用电路参考

典型的模块（如兼容 Arduino 的 Mini 电机驱动板）已集成所有外围元件（去耦电容、GND 连接），用户只需连接 VM / VCC / GND 和控制信号即可。如果自行设计 PCB，需在 VM 与 GND 之间加 10μF~47μF 电容，VM 引脚前串一保险丝或自恢复保险做短路保护。

## 单片机控制逻辑

### 硬件资源分配

TB6612FNG 需要 MCU 提供以下资源：

- **2 路 PWM 输出**（PWMA、PWMB）—— 来自定时器，用于调速
- **5 个 GPIO** —— AIN1、AIN2、BIN1、BIN2 控制方向，STBY 控制待机/唤醒

典型分配：PWMA→PA0(TIM2_CH1)，PWMB→PA1(TIM2_CH2)，方向引脚和 STBY 用任意 GPIO。

### 初始化思路

1. **PWM 定时器：** 配置一个定时器产生两路 PWM，设置预分频器和周期（ARR）确定 PWM 频率和分辨率
2. **GPIO：** 方向引脚和 STBY 都设为推挽输出，初始输出低电平
3. **启动 PWM 输出**后，电机两端的平均电压由占空比控制

### 控制逻辑

- **正转：** AIN1=H, AIN2=L，PWMA 写目标占空比
- **反转：** AIN1=L, AIN2=H，PWMA 写目标占空比
- **刹车（短路制动）：** AIN1=H, AIN2=H（电机两端短路，急停）
- **停止（惯性滑行）：** AIN1=L, AIN2=L（H 桥高阻，电机自由转动到停）
- **待机/急停：** STBY=L，所有输出高阻，电机断电

速度值由 TIM 的 CCR（比较值）控制：CCR 从 0 到 ARR，对应 0%~100% 占空比。

### 使用流程

1. 初始化 PWM 定时器和 GPIO
2. 启动 PWM 输出
3. STBY 拉高，退出待机
4. 调用方向+速度函数驱动电机
5. 需要停止时选择刹车或滑行

## 速度控制进阶

### 调速原理

电机转速与 PWM 占空比成正比。占空比 = CCR / (ARR+1) × 100%，占空比越大，电机两端平均电压越高，转速越快。ARR 决定调速分辨率（例如 ARR=999 时 CCR 可设 0~1000，精度 0.1%）。

### 缓启缓停

直接跳变占空比会导致急加速/急减速，产生机械冲击和电流尖峰。应在循环中逐步递增/递减 CCR 值，每步加适量延时来控制加速/减速斜率，实现平滑启停。

### PID 闭环调速

如果需要精确的速度控制，引入编码器测速做反馈：

```
目标转速 → PID 控制器 → PWM 占空比 → TB6612FNG → 电机 → 实际转速
                              ↑                          │
                              └── 编码器测速 ←────────────┘
```

PID 根据目标转速与实际转速的误差计算输出值，输出值为正则正转、为负则反转，绝对值作为占空比。需对输出做限幅保护。

### PWM 频率选择建议

| 频率 | 特点 | 适用场景 |
|------|------|---------|
| 1kHz~5kHz | 低速扭矩好，但可能听到线圈啸叫 | 大负载、低转速 |
| 10kHz~20kHz | 人耳基本听不到啸叫，效率适中 | **通用推荐** |
| 30kHz~50kHz | 噪音极小，但开关损耗增加 | 高速电机、噪音敏感场景 |
| >50kHz | 开关损耗明显，MOSFET 发热增大 | 一般不推荐 |
