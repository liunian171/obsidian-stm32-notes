# 前言：什么是线性变换

## 核心思想

**矩阵 = 描述向量如何运动的"指令手册"**

3Blue1Brown 的精髓：**不要把矩阵看作一堆数字，而是一个改变空间的操作。**

## 向量 vs 点

```
      y
      ^
      |    · (3, 2)
      |   /
      |  /
      | /
      +-------> x
```

向量本质就是**从原点出发的箭头**。我们用坐标 `(x, y)` 来描述它。

## 什么是线性变换？

想象整个平面（或空间）被"拉扯"，但保持两个特性：

1. **直线仍然是直线**（不打弯）
2. **原点保持不变**（不移动）

> [!tip] 类比
> 把空间想象成一块橡皮膜，你从各个方向拉伸它——网格线保持平行且等距分布。

## 关键洞察

**一个线性变换完全由两个基向量的去向决定！**

- `î = (1, 0)` → 变换后的位置
- `ĵ = (0, 1)` → 变换后的位置

这两个去向拼在一起，就是**矩阵**。

```
变换矩阵 = [ î的去向  ĵ的去向 ]
```

## 示例

假设 `î → (2, 0)`, `ĵ → (0, 3)`

```
矩阵 = [2  0]
       [0  3]
```

这个矩阵对所有向量的作用：**x 方向拉伸 2 倍，y 方向拉伸 3 倍**。

## 矩阵乘法怎么理解？

要算 `A·v`（矩阵×向量）：

> **取矩阵的列，按向量的分量进行线性组合**

```
[2  0] · [3]  = 3 × [2] + 2 × [0]  = [6]
[0  3]   [2]       [0]     [3]      [6]
```

**几何含义**：把原来向量的分量看作"倍数"，重新组合变换后的基向量。
