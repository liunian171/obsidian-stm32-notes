# 缩放与反射变换

## 缩放矩阵 (Scale)

最简单的变换：让 `î` 和 `ĵ` 在自己的方向上拉伸。

### 公式

```
S = [ s_x   0  ]
    [  0   s_y ]
```

- `s_x`：x 方向的缩放因子
- `s_y`：y 方向的缩放因子

### 图示

```
  缩放前             缩放后 (s_x=2, s_y=0.5)
  y                  y
  ^                  ^
  |  ■               |  ■
  | /                | /
  |/                 |/
  +------> x         +------> x
  单位圆变成椭圆！     x方向拉长，y方向压扁
```

### 均匀缩放

```
S = [ k  0 ]    两个方向缩放相同
    [ 0  k ]
```

这就是将向量整体放大 k 倍，相当于**标量乘法**。

## 反射矩阵 (Reflection)

"照镜子"——把空间关于某条直线翻转。

### 关于 x 轴反射

```
R_x = [ 1   0 ]     î 不动，ĵ 翻转到对面
      [ 0  -1 ]
```

```
(x, y) → (x, -y)
```

### 关于 y 轴反射

```
R_y = [ -1  0 ]     î 翻转，ĵ 不动
      [  0  1 ]
```

### 关于原点的反射（其实是旋转 180°）

```
R_origin = [ -1   0 ]
           [  0  -1 ]
```

### 关于直线 y = x 反射

```
R_y=x = [ 0  1 ]
        [ 1  0 ]
```

这相当于交换 x 和 y 坐标：`(x, y) → (y, x)`

> [!tip] 核心规律
> 反射矩阵的**行列式为 -1**。这是识别反射操作的标志。

## 缩放 + 反射的组合

```
A = [ -2   0 ]
    [  0   3 ]
```

含义：x 方向放大 2 倍 + 反射，y 方向放大 3 倍。
