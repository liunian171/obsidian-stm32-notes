# 剪切变换

## 直觉——推卡片

想象一副扑克牌，你从侧面推它——牌之间滑动，但每张牌的顶部和底部保持平行。

```
原来：             剪切后：
┌──────┐           ┌──────┐
│      │    →      │      ╱
│      │           │     ╱
│      │           │    ╱
│      │           │   ╱
└──────┘           └──╱
```

## 水平剪切 (Shear X)

```
H = [ 1   k ]
    [ 0   1 ]
```

- `î = (1, 0)` 不动
- `ĵ = (k, 1)` 向右倾斜 k 个单位

**效果**：y 坐标不变，x 坐标按 y 值剪切：

```
(x, y) → (x + k·y, y)
```

> [!example]
> 如果 k = 1，点 `(0, 1)` → `(1, 1)`，点 `(0, 2)` → `(2, 2)`
> 
> 越高的点被"推"得越远。

## 垂直剪切 (Shear Y)

```
V = [ 1   0 ]
    [ k   1 ]
```

- `î = (1, k)` 向上倾斜
- `ĵ = (0, 1)` 不动

**效果**：x 坐标不变，y 坐标按 x 值剪切：

```
(x, y) → (x, y + k·x)
```

## 剪切与面积

**剪切变换不改变面积！**

```
det(H) = det(V) = 1
```

图中的正方形变成平行四边形，但面积不变。这很像"变形但不压缩"。

> [!tip] 生活类比
> 想象一个装满水的方形鱼缸，你把一边往前推——水体变成平行六面体，但水量（面积）不变。

## 用途

- 斜体字体设计（文字倾斜）
- 图像处理中的扭曲效果
- 晶体学中的晶格变形
- 在[[6. 复合变换|复合变换]]中，剪切常用来分解更复杂的变换
