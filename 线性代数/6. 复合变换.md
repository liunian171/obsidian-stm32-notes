# 复合变换

## 核心思想

**先做 A 变换，再做 B 变换 = B·A**

> [!warning] 顺序很重要！
> 矩阵乘法**不满足交换律**：`AB ≠ BA`

## 从基向量理解

要理解 `M = BA` 这个复合变换：

1. 先对空间做 A 变换
2. 再对**结果空间**做 B 变换

对于任意向量 `v`：

```
v' = B·(A·v) = (B·A)·v
```

## 图示

```
  原始            先旋转 90°          再剪切
  ┌──┐             ┌──┐               ┌──┐
  │  │    R(90°)   │  │    H(1)       │  ╱
  │  │    ─────→   │  │    ─────→     │ ╱
  │  │             │  │               │╱
  └──┘             └──┘               └──┘
```

## 矩阵乘法公式

```
A = [ a₁₁  a₁₂ ]    B = [ b₁₁  b₁₂ ]
    [ a₂₁  a₂₂ ]        [ b₂₁  b₂₂ ]

BA = [ b₁₁a₁₁+b₁₂a₂₁    b₁₁a₁₂+b₁₂a₂₂ ]
     [ b₂₁a₁₁+b₂₂a₂₁    b₂₁a₁₂+b₂₂a₂₂ ]
```

老办法：**左行乘右列**。

但用 3Blue1Brown 的视角理解更直观：

> **B·A 的列 = B 作用于 A 的每一列的结果**

```
B·A = B·[ A的列1  A的列2 ]
     = [ B·(A的列1)  B·(A的列2) ]
```

## 为什么顺序重要？

```
旋转然后剪切 ≠ 剪切然后旋转
```

```
       旋转→剪切                   剪切→旋转
       ┌──────┐                   ┌──────┐
       │  ╱   │                   │ ╲    │
       │ ╱    │                   │  ╲   │
       │╱     │                   │   ╲  │
       └──────┘                   └──────┘
```

> [!tip] 形象记忆
> "先穿袜子再穿鞋" ≠ "先穿鞋再穿袜子"。两个操作都变了空间，后一个操作作用于已经变形的空间。

## 结合律

好消息：矩阵乘法满足**结合律**。

```
(AB)C = A(BC)
```

这让我们可以放心地做复合：

```
M = R(θ) · S · H(k)
```

你可以先算出 `S·H(k)`，再左乘 `R(θ)`，结果一样。
