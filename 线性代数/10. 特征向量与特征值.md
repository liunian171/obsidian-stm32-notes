# 特征向量与特征值

## 核心思想

对于大多数线性变换，**绝大多数向量在变换后方向都会改变**。

但某些特殊的向量——**特征向量**——它们**只被拉伸或压缩，方向不变**。

```
A·v = λ·v

v = 特征向量（方向不变）
λ = 特征值（拉伸倍数）
```

## 几何直觉

拿"水平剪切"举个例子：

```
H = [ 1  1 ]
    [ 0  1 ]
```

```
  原始                   剪切后
  ┌──┐                   ┌──┐
  │↑ │                   │↑  ╱
  │⤡ │                   │  ⤡
  │  │                   │   ╲
  └──┘                   └───┘

  ↑ 方向不变（特征向量）
  ⤡ 方向改变（普通向量）
```

在这个变换中：
- x 轴上的向量 `(1,0)` → 仍为 `(1,0)`，**方向不变** → 特征向量，λ = 1
- 其他方向的向量都被"推歪了"

## 另一种视角

> **特征向量所在的直线（特征空间）在变换后仍然是同一条直线。**

```
       ·       变换后        ·
       v       ────→        λv
       │                    │
  ─────┴─────          ─────┴─────
  特征空间                   特征空间仍在原直线上
  （一条过原点的线）          （只是被拉伸/压缩）
```

## 怎么找特征向量？

```
A·v = λ·v
A·v - λ·v = 0
(A - λI)·v = 0
```

这是一个齐次线性方程组。它有**非零解**的条件：

```
det(A - λI) = 0   ← 特征方程
```

> [!note] 特征方程
> 把行列式为 0 的方程展开，得到 λ 的多项式（特征多项式）。
> 解这个多项式就得到特征值 λ，再代回去求特征向量。

### 示例

```
A = [ 3  0 ]
    [ 0  2 ]
```

特征方程：

```
det([ 3-λ   0  ]) = (3-λ)(2-λ) = 0
    [  0   2-λ ]
```

特征值：λ₁ = 3, λ₂ = 2

特征向量：
- λ₁=3：`v₁ = (1, 0)`（x 方向被拉伸 3 倍）
- λ₂=2：`v₂ = (0, 1)`（y 方向被拉伸 2 倍）

## 特征基

如果特征向量足够多（能张成整个空间），我们可以用它们做**基**。

在这个基下，变换矩阵会变成**对角矩阵**：

```
D = [ λ₁  0  ...  0  ]
    [  0  λ₂ ...  0  ]
    [  ⋮   ⋮   ⋱   ⋮  ]
    [  0   0  ... λn ]
```

> [!tip] 对角化的意义
> 在特征基下，复杂的变换简化为**各坐标轴独立的缩放**。
> 
> 这类似于：从复杂的角度切换到"本色出演"的角度来看待变换。

## 应用场景

| 领域 | 用途 |
|------|------|
| 主成分分析 (PCA) | 寻找数据方差最大的方向（特征向量） |
| 搜索引擎排名 | PageRank 的核心是特征向量 |
| 量子力学 | 薛定谔方程的解就是特征向量 |
| 图像压缩 | 特征分解 → 保留最大特征值 → 压缩 |
| 振动分析 | 系统的固有频率 = 特征值 |
