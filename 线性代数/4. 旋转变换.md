# 旋转变换

## 视觉直觉

顺时针旋转 θ 角度时，看看基向量发生了什么：

```
î = (1, 0)  →  (cosθ, -sinθ)
ĵ = (0, 1)  →  (sinθ,  cosθ)
```

## 旋转矩阵公式

```
R(θ) = [ cosθ   sinθ ]
       [ -sinθ  cosθ ]
```

> [!note]
> 这个矩阵做的是 **逆时针** 旋转 θ 角度。

## 为什么是这个形式？

从单位圆上看：

```
旋转前：         旋转 θ 后：
î = (1, 0)      î' = (cosθ, sinθ)
ĵ = (0, 1)      ĵ' = (-sinθ, cosθ)
```

```
      y                     y
      ^                     ^  ĵ' ·
      |   ĵ                 |   /   î'
      |   |                 |  /
      |   |                 | /
      +-------> x           +------------> x
        î
```

## 特殊角度

| θ | cosθ | sinθ | 矩阵 |
|---|------|------|------|
| 0° | 1 | 0 | `[1 0; 0 1]` (恒等) |
| 90° | 0 | 1 | `[0 1; -1 0]` |
| 180° | -1 | 0 | `[-1 0; 0 -1]` |
| 270° | 0 | -1 | `[0 -1; 1 0]` |

## 顺时针 vs 逆时针

如果想顺时针旋转 θ：

```
R_cw(θ) = [ cosθ   -sinθ ]
          [ sinθ    cosθ ]
```

注意：这个矩阵就是 `R(-θ)`，也等于 `R(θ)⁻¹`（转置）。

## 旋转不改变大小

旋转是**正交变换**：

- 向量长度不变：`||R·v|| = ||v||`
- 向量间的角度不变
- 行列式为 `+1`（纯旋转）或 `-1`（含反射）
