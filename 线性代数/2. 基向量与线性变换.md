# 基向量与线性变换

## 什么是基向量？

一个二维平面中，最常用的是**标准基**：

```
î = (1, 0)    ĵ = (0, 1)
```

平面上的任何向量都可以表示为它们的**线性组合**：

```
v = x·î + y·ĵ
```

> [!info] 基的严格定义
> 一组基是一组线性无关的向量，它们能"张成"整个空间——即空间中的任何向量都可以唯一表示为它们的线性组合。

## 线性变换的"秘诀"

对于一个线性变换，我们只需要知道**两个基向量去了哪里**。

```
A = [ a  b ]     î = (a, c)
    [ c  d ]     ĵ = (b, d)
```

> **矩阵的每一列，就是基向量变换后的位置。**

## 为什么这样就能代表所有向量？

因为线性变换保持线性组合关系不变：

```
v' = T(v) = T(x·î + y·ĵ)
         = x·T(î) + y·T(ĵ)
         = x·(a,c) + y·(b,d)
         = (ax + by, cx + dy)
```

这正是矩阵乘法的规则！

```
[ a  b ] [ x ]   =   [ ax + by ]
[ c  d ] [ y ]       [ cx + dy ]
```

## 非标准基

你并不一定要用标准基。比如：

```
u = (1, 1)    v = (-1, 1)
```

这也是⼀组基。在这个坐标系下：

- `(1, 0)` 在标准基中表示 `(1, 1)` 在这个新基中
- 同一个向量，在不同基下有不同的坐标

> [!tip] 记忆
> **向量是客观存在的箭头，坐标则是我们选的参考系下的描述。** 改变基 = 改变参考系。

## 二维到三维

三维的情况完全相同，只是多了一个基向量 `k̂`：

```
旋转矩阵 = [ î_x  ĵ_x  k̂_x ]
          [ î_y  ĵ_y  k̂_y ]
          [ î_z  ĵ_z  k̂_z ]
```
