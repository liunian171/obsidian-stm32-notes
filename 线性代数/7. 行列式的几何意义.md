# 行列式的几何意义

## 一句话概括

> **行列式 = 线性变换改变"面积"（体积）的倍数**

## 直观理解

取 1×1 的单位正方形（面积 = 1），对它做线性变换，得到的平行四边形面积就是 `|det(A)|`。

```
                   旋转、缩放后：
  原始 面积=1       面积=det(A)
  ┌──┐              ┌──┐
  │  │     →        │  │ (面积变了，但仍是平行四边形)
  └──┘              └──┘
```

### 二维

```
det([ a  b ]) = ad - bc
    [ c  d ]
```

**几何含义**：基向量 `î` 和 `ĵ` 张成的平行四边形面积。

```
det > 0    方向不变（î 在 ĵ 右侧）
det < 0    方向翻转（î 在 ĵ 左侧，发生了反射）
det = 0    降维（面积压没了→矩阵不可逆）
```

### 三维

行列式 = 平行六面体的体积。

```
det = 0 → 三维压缩到二维→体积为0 → 降维了
```

## 关键性质

| 性质 | 解释 |
|------|------|
| `det(I) = 1` | 恒等变换不改变面积 |
| `det(AB) = det(A)·det(B)` | 复合变换的行列式 = 各变换行列式的积 |
| `det = 0 ⟺ 不可逆` | 降维了就没法"恢复" |
| `det < 0` | 空间被翻转（左右镜像） |

## 用行列式理解"可逆"

> [!danger] det = 0 意味着什么？
> 如果行列式为 0，变换把平面压缩为一条线或一个点。这导致：
> - 多个不同的向量被映射到同一个点
> - 无法进行逆变换（信息丢失了）
> - 矩阵不可逆

## 面积计算示例

```
A = [ 3  1 ]
    [ 0  2 ]

det(A) = 3×2 - 1×0 = 6
```

含义：使用 A 变换后，任何区域的面积都变为原来的 6 倍。
