# 逆变换与逆矩阵

## 核心思想

**逆变换 = 把变换后的空间"还原"的操作**

```
A⁻¹ · A = I    （先 A 变换，再 A⁻¹ 变换 = 什么都没做）
```

## 几何理解

```
   原始                               逆变换恢复
   ┌──┐     旋转 90°      ┌──┐      旋转 -90°      ┌──┐
   │  │     ─────→        │  │      ──────→         │  │
   │  │                   │  │                      │  │
   └──┘                   └──┘                      └──┘
              A·v = v'                   A⁻¹·v' = v
```

## 2×2 逆矩阵公式

```
A = [ a  b ]    →    A⁻¹ =  1    [  d  -b ]
    [ c  d ]               ad-bc  [ -c   a ]
```

其中 `ad - bc = det(A)`。

> [!tip] 
> 公式中 1/det(A) 意味着：**行列式为 0 时没有逆矩阵**——因为 1/0 没有意义，几何上是因为信息丢失不可恢复。

## 几何条件

什么时候一个变换可逆？

| 条件 | 几何含义 |
|------|----------|
| `det(A) ≠ 0` | 面积没有被压缩到 0 |
| 不降维 | 二维 → 二维，三维 → 三维 |
| 一一映射 | 不同向量不会映射到同一个点 |

## 线性方程组的几何解释

解方程组 `Ax = b` 的几何含义：

> 寻找一个向量 x，使得经过 A 变换后，它落在向量 b 的位置。

```
            A·x = b
    原始     →       变换后
        x             b
       · ···········> ·
```

如果有 `A⁻¹`：

```
x = A⁻¹·b
```

## 三种情况

| det(A) | 解的情况 | 几何解释 |
|--------|----------|----------|
| ≠ 0 | 唯一解 | 一个 x 对应一个 b |
| = 0, b 在像空间内 | 无穷多解 | 一整条线压缩到一个点 |
| = 0, b 不在像空间内 | 无解 | b 在降维后的空间之外 |
