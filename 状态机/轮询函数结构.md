// ===================== 第1段：定义本模块的所有状态（枚举）=====================
// 把这个模块能做的事，全部拆成独立状态，起名见名知意
`typedef enum`
`{`
    `状态1,`
    `状态2,`
    `状态3,`
    `// ... 按需加`
`} 模块名_STATE;`

// ===================== 第2段：本模块专属的状态变量（核心记忆）=====================
// 静态变量：只属于这个函数，记住「当前在哪个状态」，轮询不会丢
```
static 模块名_STATE 当前状态变量 = 初始状态;
```

// ===================== 第3段：可选 → 本模块辅助变量（计时/标志，无delay用）============
```
static uint16_t 计时变量 = 0;  // 用来做延时、闪烁、防抖，绝对不用delay()
static uint8_t 标志变量 = 0;
```

// ===================== 第4段：轮询函数主体（switch 核心）=====================
```
void 模块名_FSM(void)  // 函数名固定：模块名+FSM(状态机)
{
    // 1. 直接跳转到当前状态执行
    switch(当前状态变量)
    {
        case 状态1:
            // 只做状态1该做的事
            // 判断切换条件 → 赋值修改状态变量
            break;

        case 状态2:
            // 只做状态2该做的事
            // 判断切换条件 → 赋值修改状态变量
            break;

        default:
            // 容错：跑飞了自动回初始状态
            当前状态变量 = 初始状态;
            break;
    }

    // 函数执行完毕，立刻返回！不卡住CPU
}
```
